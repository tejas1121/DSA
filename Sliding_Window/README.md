🔥 Core Idea
right → expands window
left → shrinks window

👉 You dynamically adjust window based on condition

🧠 Template (VERY IMPORTANT — memorize)
left = 0

for right in range(len(arr)):

    # expand window (include arr[right])

        # shrink window
        left += 1
