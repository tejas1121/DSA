Use it when:

You have arrays / strings
You need to compare elements from both ends
Problem involves:
palindrome
pair sum
reversing
sorted arrays



Type 1: Opposite Direction (Left ↔ Right)
📌 Structure:
left = 0
right = len(arr) - 1

while left < right:
    # logic
    left += 1
    right -= 1
📌 Used in:
Palindrome
Pair sum in sorted array
Reverse array
🔹 Type 2: Same Direction (Fast–Slow) ⭐ MOST IMPORTANT
📌 Structure:
i = 0  # slow
for j in range(len(arr)):  # fast
    if condition:
        arr[i] = arr[j]
        i += 1
📌 Idea:
j → explores everything (fast)
i → builds result (slow)

👉 This is what you just used in remove duplicates

🔹 Type 3: Fast–Slow (Different Speed)
📌 Structure:
slow = head
fast = head

while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
📌 Used in:
Linked list cycle
Middle of list

(We’ll cover later)

🔹 Type 4: Sliding Window (Advanced Two Pointer)
📌 Structure:
left = 0

for right in range(len(arr)):
    # expand window

    while condition:
        left += 1

👉 Used for:

Subarrays
Substrings
